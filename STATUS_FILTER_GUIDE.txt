===========================================
STATUS FILTER IMPLEMENTATION GUIDE
===========================================

This guide shows exactly what changes to make in each file to add status filtering.
All changes filter out posts where status is "disabled" or "expired", showing only "active" posts.

===========================================
1. WORKERS.JS
===========================================
Location: Line 473 (after "const filtered = withDistance.filter(worker => {")

ADD THESE 3 LINES:
      // Status filter - only show active posts
      if (worker.status && worker.status !== "active") return false;

BEFORE the existing line:
      // Search

===========================================
2. SERVICES.JS  
===========================================
Location: Line 710

REPLACE THIS LINE:
      if (service.status === "expired") return false;

WITH THESE 2 LINES:
      // Filter out expired and disabled posts
      if (service.status && service.status !== "active") return false;

===========================================
3. ADS.JS
===========================================
Find the line that starts with: "const filtered"
Then find the ".filter(ad => {" line
Right after the opening brace "{", ADD:

      // Status filter - only show active posts
      if (ad.status && ad.status !== "active") return false;

===========================================
4. FAVORITES.JS
===========================================
Find each ".filter(" section for workers, services, and ads
After each filter's opening brace, ADD:

      // Status filter - only show active posts  
      if (item.status && item.status !== "active") return false;

===========================================
TESTING
===========================================
After making these changes:
1. Save all files
2. Check for compilation errors
3. Test by:
   - Creating a post
   - Disabling it from Profile page
   - Verifying it disappears from main pages and favorites
   - Enabling it again
   - Verifying it reappears
